2 Kelas x 3 SKS = 6

Mengajar Mata Kuliah Sistem Tersebar
1 kelas x 3 SKS = 3

Untuk 10 SKS Pertama :
9 x 1(sks/semester) = 9 SKS

Untuk 2 SKS Kedua :
0 x 0.5(sks/semester) = 0 SKS

Total SKS = 9 SKS
